Parent Div



stopPropagation()

Purpose: Prevents the event from bubbling up to parent elements.
Effect: Once called, the event will not be dispatched to any parent elements. However, it will still allow other event listeners on the same element to execute.


stopImmediatePropagation()

Purpose: Prevents the event from bubbling up to parent elements and stops any other event listeners on the same element from being executed.
Effect: Once called, it will halt any further event listeners on the current target element from running, in addition to preventing the event from bubbling up.
